# Prédiction du rendement du maïs au Togo (1961–2024)

## Objectif
Analyser l'évolution du rendement agricole du maïs au Togo 
et construire un modèle de prédiction avec Python.

## Données
Source : FAOSTAT (FAO, Nations Unies)
Période : 1961 à 2024 — 64 observations

## Outils utilisés
- Python, Pandas, Matplotlib, Scikit-learn
- Google Colab

## Résultats
- MAE : 173.92 kg/ha
- R² : 0.36
- Tendance haussière confirmée sur 64 ans

## Limites et améliorations
Le modèle utilise uniquement l'année comme variable.
Des améliorations sont possibles en ajoutant : 
pluviométrie, engrais, politiques agricoles.
